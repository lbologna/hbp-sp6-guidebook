Morphology VisualizationÂ¶

The Morphology Visualization Use Case provides the possibility to visualize a 3D neuronal morphology in a browser window.

  1. This Use Case is in the Online Use Cases/Morphology Analysis.

  2. At the Morphology Analysis Use Case, select single cell morphology from the HBP data.

  3. The single cell morphology will open in a new browser window, ready for visual inspection.
